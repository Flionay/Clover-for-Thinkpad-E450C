## E450c Clover for Mojave 10.14.3
E450c黑苹果完美clover

用的是[zhangxuan](https://github.com/zhangxuan1340/Hackintosh_E450C)的引导文件，非常感谢他！

电量显示完美，感谢另一个大哥的帮助，虽然我还不知道他的名字。

声卡都可以驱动，利用AppleHDA patcher，按照黑苹果屋的方法可以驱动很完美！声卡型号是Conexant CX20752.

HDMI可以输出，完美当作台式机用。

折腾黑苹果终于告一段落了。
